Main controller function.

This node is the actual controller of the robot, it keeps track of position, sets velocities, and asks for new targets. Node initialization; Initialiazes a client to a service /goal for receiving a new target. Initializes a subscriber to /odom to retrieve robot pose, and a publisher to /cmd_vel to set robot velocity.

void positionCallback ( const nav_msgs::Odometry::ConstPtr &  msg)

Callback called by subscriber to topic /odom.

This Callback receives the robot pose from /odom, computes the distance from the goal. If the goal is reached a new target is asked by a client. Robot velocity is adjusted with respect to distance.
